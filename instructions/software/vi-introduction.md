## 简介

vi 是 Linux 系统下的命令行文本编辑器, vim 是它的一个扩展

随着一系列的神器的出现, 现在服务端上直接使用命令行修改文件以及是很少见的了, 像 vs code 之类的 ide 或者 ssh 客户端等直接可以通过远程方式打开服务器文件并修改

当然多学习一些相应的操作也能应付不时之需, 这里简单介绍一下 vi 的使用方法, 并对经常使用的快捷键进行说明, 还有提供相应的介绍文档
